# Uma Aventura Congelante — guia visual

- Azul profundo `#082F55`: texto e superfícies escuras.
- Azul glacial `#168FC5`: ações e destaques.
- Neve `#F3FBFF`: fundo.
- Gelo `#DFF5FF`: superfícies secundárias.
- Lilás polar `#B9A7E8`: detalhes mágicos.
- Marcellus: títulos de conto de fadas; DM Sans: interface e leitura.
